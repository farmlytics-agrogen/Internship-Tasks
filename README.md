# GPT-2 Text Generation

## Description
This project demonstrates basic text generation using the GPT-2 model from Hugging Face Transformers.

## Technologies Used
- Python
- Google Colab
- Hugging Face Transformers
- GPT-2

## How to Run
1. Install:
   pip install transformers torch
2. Run the notebook.
3. Enter a prompt.
4. The model generates text based on the prompt.

## Sample Prompt
Artificial Intelligence is

## Sample Output
Artificial Intelligence is changing the way people work, learn, and communicate...
